Detailed Description

Utilities for reading and writing 2D grid data in VTK STRUCTURED_GRID format.

This header provides functions to export a flattened 2D grid to a VTK file for visualization, and to read grid data and coordinates from a VTK file in STRUCTURED_GRID format.
