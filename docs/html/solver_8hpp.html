Detailed Description

Header file for the Solver class that implements various numerical methods for solving the Laplace equation.

This header defines the Solver class which provides multiple implementations for solving the Laplace equation using different parallelization strategies including serial execution, OpenMP, MPI, and hybrid MPI+OpenMP approaches. The class also supports direct solving methods using Eigen's sparse matrix decomposition.

The solver supports:

  • Jacobi iterative method with various parallelization strategies
  • Direct solving using Schwarz domain decomposition
  • Boundary condition specification through function objects
  • Error computation and convergence monitoring
  • VTK output for visualization
