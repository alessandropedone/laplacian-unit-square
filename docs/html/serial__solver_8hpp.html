This file contains the declaration of the SerialSolver class, which implements an iterative solver for a given equation. The class provides methods to set the boundary conditions, initial guess, exact solution, and right-hand side of the equation.
It also includes a method to print the computed solution.

The class is designed to be used in a serial computing environment. It is not optimized for parallel computing and does not use any parallel computing libraries or techniques.

The class is intended to be used as a comparison for more complex solvers that may incorporate parallelism or other advanced features.
