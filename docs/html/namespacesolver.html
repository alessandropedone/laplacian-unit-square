Detailed Description

Namespace containing classes and functions for solving partial differential equations.

Contains utilities and data structures for distributed numerical simulations.

This namespace provides implementations for solving the Laplace equation using various numerical methods and parallelization strategies. It includes serial and parallel implementations using OpenMP, MPI, and hybrid approaches.

The main class Solver provides multiple solution methods:

  • Serial Jacobi iteration
  • OpenMP parallelized Jacobi iteration
  • MPI parallelized Jacobi iteration
  • Hybrid MPI+OpenMP Jacobi iteration
  • Direct solution using domain decomposition (Schwarz method)

The solver supports:

  • Custom boundary conditions on all four sides
  • User-defined right-hand side functions
  • Error computation against exact solutions
  • VTK output for visualization
  • Configurable convergence criteria

This namespace provides MPI-based broadcasting functionality and parameter management for parallel Poisson equation solvers. It includes utilities for distributing simulation configuration across multiple processes in a distributed computing environment.
